# Responsive Banking UI
# Aim:
To create a responsive banking interface with deposit and withdrawal functionality using HTML, CSS, and JavaScript.
# Objective
         Design UI using CSS Flexbox
         Implement Deposit & Withdraw using JavaScript
         Validate user input and display messages
         Make layout responsive using media queries

# Hardware / Software Requirements
         Hardware: PC / Laptop
         Software: VS Code, Web Browser (Chrome)

# About the Program

   This program displays a bank balance and allows the user to deposit or withdraw money.
   JavaScript handles balance updates and input validation, while CSS Flexbox ensures a responsive layout.

# Files Used
        index.html – Structure of the banking UI
        style.css – Styling and responsive layout
        script.js – Logic for deposit, withdraw, and validation

# Working
       Initial balance is displayed on screen
       User enters amount
       Deposit: Adds amount to balance
# Withdraw: Subtracts amount if sufficient balance
# Invalid input shows an error message

# Output
    A responsive banking interface showing balance, input field, deposit & withdraw buttons with validation feedback.

# Conclusion
    This experiment demonstrates Flexbox layout, DOM manipulation, and basic state management using JavaScript.
